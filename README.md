# Barbell OS

A Naval / Taleb-coded one-page system for allocating resources with asymmetric upside.

**Principle:** Protect downside, seek convex upside, remove fragility (via negativa).

## What it does
- Choose a domain (Money / Time / Health / Career)
- Set Safe % (70–95)
- Write:
  - Safe plan (boring compounding)
  - Convex bets (optionality)
  - Kill list (fragility removal)
- Generates a plan card + Markdown export (copy)
- Stores data locally in your browser (LocalStorage)

## Run locally
Open `index.html` in your browser.

## Deploy
Use GitHub Pages:
Settings → Pages → Deploy from branch → `main` / `root`.
